Sistema de Pedidos e Inventariado

Capacidad

El sistema cuenta con una capacidad suficiente para almacenar toda la informacion necesaria para la gestion de inventario y su funcionamiento

Funcion

No es necesario conocimientos previos para el manejo correcto de dicho sistema, solo se necesita personas que quieran aprender a usarlo

Multiplataforma

El sistema es compatible con distintos tipos de dispositivos, tanto celulares como computadores

Costo

Es un sistema de bajo costo, el cual puede ser implementado tanto como en tiendas de gran nivel como las de pequeño nivel

Acceso

El sistema cuenta con un acceso remoto y por conexion Wi-Fi, es decir puedes acceder a el desde todo lugar

Para acceder de forma remota es necesario contar con un acceso no IP, que mediante una conexion a internet hace efectivo el ingreso al sistema desde cualquier dispositivo

Otro modo de acceso es de forma por conexion Wi-Fi, la persona que quiera tener acceso debe estar cerca del servidor y conectarse a la red Wi-fi del mismo para poder ingresar a este mediante un navegador de forma local

Como utilizar el sistema?

Primero es necesario pasar por un proceso de logueo el cual requiere una cuenta con su "ID" y su "Password" para poder ingresar al sistema una vez hecha la conexion

Para poder realizar un pedido, es necesario ir al link de realizar un pedido, en el cual una vez elegido todos los productos necesarios dar click en aceptar

De igual manera para poder revisar el inventario o los productos ordenados, es necesario contar con los anteriores requerimientos
